Функции

Работа с радаром
ScanRadar ( angle# )
Сканирует пространство под углом angle безотносительно от направления движения бота.
Если, например, angle=0, то пространство сканируется все время в одном и том же направлении.
Область сканирования - сектор. При этом, препятствия засекаются только по направлению "главного луча", а бонусы и другие боты - по всему сектору.
1 - препятствие, засеченное радаром ( т.к. попал под "главный луч" ).
2 - препятствие, не засеченное радаром ( т.к. не попадает под "главный луч" ).
3 - бонус, засеченный радаром ( т.к. попал в сектор сканирования ).
4 - "главный луч".

PointRadar ( entity% )
Направляет радар на объект entity ( указатель ).


Работа с объектами

EntityX ( entity% ), EntityY ( entity% ), EntityZ ( entity% )
возвращает соответствующие координаты объекта в пространстве.
EntityPitch ( entity% ), EntityYaw ( entity% ), EntityRoll ( entity% )
возвращает соответствующие углы объекта в пространстве.
EntityVisible ( src_entity%, dest_entity% )
определяет видим ли один объект другим.
EntityDistance ( src_entity%, dest_entity% )
возвращает расстояние между объектами.
DeltaYaw ( src_entity%, dest_entity% )
угол, на который надо повернуть объект src_entity по оси OY, чтобы направить его на dest_entity.
На рисунке src_entity - бот, движущийся по направлению вертикальной стрелки, dest_entity - некий объект.
Функция DeltaYaw возвращает угол "фи".
DeltaPitch ( src_entity%, dest_entity% )
угол, на который надо повернуть объект src_entity по оси OX, чтобы направить его на dest_entity.
На рисунке src_entity - это ствол бота, dest_entity - некий объект.
Функция DeltaPitch возвращает угол "лямбда".
PositionPivot ( x#, y#, z# )
позиционировать виртуальную точку Pivot в заданные координаты.
TypeEntity( entity )
возвращает тип объекта. Враг 1, Друг 2, Препятствие 3-4, Снаряды 5, Ракеты 6, Ремонт 7
Lives( bot )
возвращает количество жизней бота (в том числе и вражеских)


Работа с коллизиями ( столкновениями )

CollisionX ( entity% ), CollisionY ( entity% ), CollisionZ ( entity% )
возвращает соответствующую координату последнего места столкновения объекта.
CollisionNX ( entity% ), CollisionNY ( entity% ), CollisionNZ ( entity% )
возвращает соответствующую компоненту нормали последнего места столкновения объекта.


Математика

Abs ( x# )
возвращает модуль числа x#.
Sqr ( x# )
возвращает квадратный корень числа x#.
Sgn ( x# )
возвращает знак числа x#.
Sin ( x# )
возвращает синус угла x#.
Cos ( x# )
возвращает косинус угла x#.
ASin ( x# )
возвращает арксинус числа x#.
ACos ( x# )
возвращает арккосинус числа x#.
Min ( a#, b# )
возвращает минимальное число.
Max ( a#, b# )
возвращает максимальное число.
Rnd ( a#, b# )
возвращает случайное дробное число.
Rand ( a#, b# )
возвращает случайное целое число.
ATan2 ( x#, y# )
возвращает угол от оси OY до точки (x, y). В левой полуплоскости от оси OY значения Atan2 отрицательные, в правой - положительные.
Atan2(10, 10) = 45
Atan2(-10, -3) = -106.699
Эта функция крайне полезна, когда есть два обьекта, и нужно повернуть один к другому. :
ATan2 ( y2-y1, x2-x1 )
выдаст правильную ориентацию для объекта 1 ( c координатами (x1, y1) ), чтобы он был направлен на объект 2 ( c координатами (x2, y2) ).
